# Console App Agent Guidelines''## Brand Theme & Visual Design Policies''### Flat Design Only'- No gradients.'- No drop-shadows (unless totally flat/hard).'- No borders/outlines on containers or regions.''### Contrast & Separation'- Avoid thin lines for separating regions.'- Use varying background colors on flat rectangles with rounded corners to indicate regions and groupings.'- Use contrast carefully; avoid high contrast.'  - Background black should not be fully black (use dark gray).'  - Foreground white should not be fully white.'  - Use a limited set of official colors: not-black, not-white, and 2-3 muted colors.''### Color System'- Themes use Radix Colors (Cool, Neutral, Warm).'- Support both Light and Dark modes.''### Typography & Layout'- Refined elegance, modern Bauhaus-inspired, Apple's modern minimalist Art Deco.''### Spacing Rules'- Use a single spacing unit `P3` for layout.'- Outer padding: `P3`.'- Gaps between elements: `P3`.'- Internal padding: `P3`.'- Avoid additional spacing tiers.'
